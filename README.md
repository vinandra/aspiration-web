## Pengaturan-Bot

1. Buka `.env_example` ubah nama menjadi `.env`
2. Instal dependensi yang diperlukan composer install
3. setting .env DB mysql
4. Anda dapat memulai

```sh
$ php artisan migrate
```

6. Anda dapat menjalankan di local server

```sh
$ php artisan serve
```
